Actividad 1 (Valor 10 puntos) Investiga en diferentes sitios web y describe cada tema:

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje que se usa para estilizar y diseñar páginas web.

2. Funcionamiento Básico de CSS

CSS controla cómo se muestra el contenido en un sitio web, aplicando estilos como colores, fuentes y espaciados a los elementos HTML.

3. Incluir CSS en un archivo HTML.

Se puede incluir CSS de tres maneras: en línea (con el atributo style), interno ( con la etiqueta < style > dentro de < head > ) o externo (mediante un archivo .css enlazado con < link > ).

4. Definición de una regla CSS.

Una regla CSS es un bloque de código que define los estilos para uno o más elementos HTML.

5. Elemento de una regla CSS.

Una regla CSS es un bloque de código que define los estilos para uno o más elementos HTML.

•Regla:

Conjunto de estilos aplicados a un elemento.

•Selector:

Define el elemento a estilizar.

•Bloque de declaración:

Contiene las propiedades y valores.

•Propiedad:

Característica a modificar (como color).

•Valor:

Especifica el estilo de la propiedad.


6. Medios CSS.

Permiten aplicar estilos según el tipo de dispositivo o características (como @media para pantallas específicas).

7. Comentarios CSS.

Notas dentro del código, ignoradas por el navegador, se escriben con /* comentario */.

8. Sintaxis de la definición de las propiedades CSS.

Se escribe propiedad: valor; dentro de una regla CSS.
